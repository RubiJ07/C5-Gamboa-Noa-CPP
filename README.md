# C5-Gamboa-Noa-CPP

Guia teorica completa de Estructuras de Datos y Programacion Avanzada en C++ (C++17+).

Este repositorio contiene la implementacion completa en C++ de los siguientes temas:

- Pilas (Stack)
- Colas (Queue)
- Clases y punteros a miembros
- Aritmetica de punteros
- Arboles B (B-Trees)
- Templates variadicos y ForEach
- Arboles Binarios de Busqueda (BST)

Todos los codigos fueron desarrollados de acuerdo a la guia teorica proporcionada en clase y cumplen con el estandar C++17 o superior.
